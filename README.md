# MyWebsite
If you're attempting to develop this project further for your own purposes you will need to have the following installed or added as a dependency:
1. Node.js: This will give you a node_modules folder
2. expo: This is the library we use for real time updates to the DOM
3. expo-status-bar: Allows us to access/hide the status bar (who would have thought)
4. react: Creates user interfaces
5. react-native: This is the primary renderer we use for DOM manip
6. react-native-game-engine: Gives us access to time
7. matter-js: Simulates 2D physics in the browser
